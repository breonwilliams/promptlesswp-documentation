# Plugin Settings

Configuration options available in Promptless WP.

## What This Is

Settings for AI integration, data management, and global design.

## Available Settings Pages

1. **Promptless WP → AI Settings** - OpenAI API configuration
2. **Promptless WP → Data Management** - Backup and data options
3. **Promptless WP → Cache Management** - Clear plugin cache
4. **Promptless WP → Reusable Elements** - Text snippets

## AI Settings

**Setup**:
1. Go to Promptless WP → AI Settings
2. Enter OpenAI API key
3. Click "Test Connection"
4. Save settings

**Available Options**:
- API key field
- Model selection (GPT-4o)
- Connection test button

**Security**:
- Key stored in WordPress database
- Only admins can access
- Set usage limits at OpenAI

## Data Management

**Options**:
- **Delete on uninstall**: Choose to keep or remove data
- **Clear all data**: Emergency reset button

**Data Storage**:
- Sections stored as post meta
- Included in WordPress backups
- Standard export/import works

## Global Design Settings

**Access**: Click settings icon in editor toolbar

**Options**:
- Brand colors
- Font families
- Section spacing
- Light/dark themes
- Smart accessibility colors

Changes apply to all sections.

## Cache Management

**When to Clear Cache**:
- After updating global settings
- If sections not displaying correctly
- After major theme changes

**How to Clear**:
1. Go to Promptless WP → Cache Management
2. Click "Clear All Caches"


## Common Problems

**API Not Working**:
- Check API key is correct
- Test connection button
- Verify OpenAI account active

**Settings Not Saving**:
- Check user permissions
- Clear browser cache
