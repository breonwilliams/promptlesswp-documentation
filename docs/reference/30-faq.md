# FAQ

Common questions about Promptless WP.

## Getting Started

**What is Promptless WP?**
WordPress plugin for creating page sections with optional AI content generation from documents.

**Do I need coding knowledge?**
No. Visual editor with drag-and-drop sections.

**Works with my theme?**
Yes. Compatible with any WordPress theme. Use Canvas mode for full control or Full Width to keep theme header/footer.

## Requirements

**Minimum:**
- WordPress 6.0+
- PHP 7.4+
- 128MB memory

**For AI:**
- OpenAI API key
- Internet connection

**Installation:**
1. Upload plugin ZIP via WordPress admin
2. Activate plugin
3. Optional: Add OpenAI API key for AI features

**OpenAI Costs:**
Typically $5-20/month based on usage. Monitor at OpenAI dashboard.

## Using the Plugin

**Add sections:**
1. Edit page in WordPress
2. Click "Activate Promptless WP" in meta box
3. Click "Edit with Promptless WP"
4. Add sections in editor

**Limits:**
- Free: 2 pages total
- Premium: Unlimited pages
- Recommended: 10-15 sections per page

**Features:**
- Drag to reorder sections
- Duplicate sections
- Works on pages (not posts)

## AI Features

**How it works:**
Upload Word/PDF documents → AI extracts content → Generates page sections

**Supported uploads:**
- Word documents (.docx)
- PDFs
- Text files

**AI Notes:**
- Content is editable after generation
- Uses OpenAI GPT-4o model
- Don't include sensitive data
- Works in multiple languages

## Design

**Global Settings:**
Click settings icon in editor toolbar to set colors, fonts, spacing.

**Section Options:**
- Theme: Light/Dark
- Layout variants per section type
- Custom background colors

**Features:**
- Smart Accessibility Colors (automatic WCAG compliance)
- Responsive design (mobile-first)
- No built-in CSS editor

## Content Management

**User Access:**
Admins and Editors can use Promptless WP.

**Backup:**
Sections stored as post meta - included in WordPress backups.

**Deactivation:**
Content stays in database but won't display until reactivated.

## Performance

**Speed:**
Clean HTML output. Works with caching plugins.

**SEO:**
Semantic HTML. Compatible with Yoast/Rank Math.


## Troubleshooting

**Sections not showing:**
- Use Canvas or Full Width display mode
- Clear cache
- Check browser console

**Editor won't load:**
- Clear browser cache
- Disable other plugins
- Use Chrome/Firefox/Safari

**AI not working:**
- Check API key
- Verify OpenAI account active
- Check usage limits

**Upload issues:**
- Check file size/format
- Verify server limits
