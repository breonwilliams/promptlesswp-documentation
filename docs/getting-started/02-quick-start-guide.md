# Quick Start Guide

Build your first page with Promptless WP in 15 minutes.

## What This Is

A step-by-step walkthrough to create an About Us page with hero, features, and testimonial sections.

## When to Use This

- First time using Promptless WP
- Want to learn the basic workflow
- Need a quick example to follow

## Step 1: Create a New Page

1. Go to Pages â†’ Add New
2. Enter "About Us" as title
3. Scroll to "Promptless WP" meta box

## Step 2: Activate the Builder

1. Click "Activate Promptless WP"
2. Choose display mode:
   - **Canvas**: No header/footer
   - **Full Width**: With theme header/footer (recommended)
   - **Shortcode**: Manual placement with [promptless_wp]
3. Click "Edit with Promptless WP"

## Step 3: Editor Overview

- **Left Sidebar**: Add/edit sections
- **Center**: Live preview
- **Right Sidebar**: Global design settings
- **Top Toolbar**: Save and preview

## Step 4: Add a Hero Section

1. Click "+" in left sidebar
2. Select "Hero"
3. Fill in the fields:
   - Eyebrow: "Welcome to Our Company"
   - Heading: "We Create Amazing Experiences"
   - Content: "For over 10 years, we've been helping businesses grow."

Preview updates automatically as you type.

## Step 5: Add Buttons

1. Find "Buttons" in left sidebar
2. Edit first button:
   - Text: "Get Started"
   - URL: "/contact"
   - Style: Primary
3. Add second button:
   - Text: "Learn More"
   - URL: "/about"
   - Style: Secondary

## Step 6: Add Features

1. Click "+" to add new section
2. Choose "Features"
3. Set heading: "Why Choose Us?"
4. Edit the three feature cards:
   - Card 1: "Expert Team" 
   - Card 2: "Fast Delivery"
   - Card 3: "24/7 Support"
5. Add descriptions and icons for each

## Step 7: Add Testimonials

1. Click "+" and choose "Testimonials"
2. Set heading: "What Our Customers Say"
3. Add two testimonials with:
   - Quote text
   - Author name
   - Role/company
   - Star rating

## Step 8: Preview and Save

1. Click "Preview" in top toolbar
2. Check desktop and mobile views
3. Click "Save Changes"
4. Return to WordPress and click "Publish"


## You're Done!

You've created a complete page with hero, features, and testimonials.

## Next Steps

- Try different section types
- Upload a document for AI generation
- Adjust global design settings

## Tips

- Save frequently
- Preview before publishing  
- Check mobile view
- Limit to 10-15 sections per page

## Common Problems

**Changes not showing**
- Save in editor AND publish in WordPress

**Can't find editor**
- Check "Promptless WP" meta box on page

**Need to edit later**
- Return to page and click "Edit with Promptless WP" again
